SRCDS x86_64 Modding Support

This page is here to compile and track x86_64 modding support by Metamod & Sourcemod. As of writing, October 4th 2024, Team Fortress 2 & Counter Strike 2 are the primary games AlliedModders aims to bring full modding support for. The work can be loosely decomposed into three parts, which you'll find below.
Mark Definition
Full support has been achieved. No more work is required except for the eventual maintenance that can arise from game updates.
Partial support has been achieved. However we're still quite far from completion. Missing features, missing reverse engineering work, missing code.
No support. Work hasn't started, or just started. It will likely take a long time before full support is achieved.

Metamod

Rephrasing what's written on the official website. Metamod sits as the mechanism between server plugins and the srcds, in order to provide a unified hooking interface called SourceHooks. It streamlines the process of loading and unloading server plugins, and severely decreases the chances of multiple server plugins conflicting.

SourceMod

SourceMod is a metamod (server) plugin that aims to expose the various game functions as well as third-parties libraries like MySQL or CURL, to a set of plugins hosted in a sandbox environment. It is the primary framework of choice when wanting to mod the srcds in a safe manner in order to provide extra functionalities such as 'admins/moderators' or 'ban database'.
